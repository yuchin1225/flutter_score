├── images

### 設置環境

#### 1. 安裝 flutter sdk

#### 2. 檢查必要安裝
```bash
    flutter doctor
```

#### 3. 授權使用
```bash
    flutter doctor --android-licenses
```

### 執行
```bash
    flutter run
```

### 打包
```bash
    flutter build apk / flutter build apk --debug
```
